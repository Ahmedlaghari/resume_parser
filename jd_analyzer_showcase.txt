# Module 2 — JD Analyzer: Input / Output Showcase

This file demonstrates the output of the JD Analyzer pipeline across 5 real-world
job description samples. Each JD tests different extraction challenges.

Pipeline per JD:
  1. text_cleaner.py       — strips noise
  2. extractor.py          — LLM extracts base fields
  3. skill_classifier.py   — LLM classifies required vs nice-to-have skills
  4. seniority.py          — LLM detects seniority level
  5. KeyBERT               — extracts top keywords
